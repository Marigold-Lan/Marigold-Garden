基于DOTS打造的高互动性卡通箱庭世界，版本是Unity 6.4。

# 规划书

### 目的：构建一个高度自由、可互动的箱庭乌托邦世界。

#### 想法：

* [ ] **核心架构采用 DOTS 技术。**
* [ ] **可联机，网络同步。**（首先需要做好可靠扩展的单机 demo 闭环）
    * [ ] 确定性物理，定点数
* [ ] **玩法：**
    * [ ] 美丽卡通箱庭世界。
    * [ ] 老鼠种族大繁殖事件，老鼠群采用流场寻路。
    * [ ] 广播使得人群集合。
    * [ ] 门可开。
    * [ ] 窗户可打碎。
    * [ ] 大量 npc 的 AI 行为，可与环境进行交互，例如，可以坐下休息，可以睡觉，可以工作，可以被老鼠群惊扰后窜逃，可以在下雨天进入室内躲雨。
    * [ ] 可程序化生成的建筑、地形、NPC。
    * [ ] 大量其他可交互物品。
* [ ] **完整工具链开发，包括各种编辑器扩展与 Excel 导表工具**
* [ ] **涌现式化学/物理交互战斗引擎**
* [ ] **空间划分（化学交互和流场都极度依赖邻居搜索，基于 NativeHashMap 的空间哈希 (Spatial Hash) 或 八叉树 (Octree)。）**
* [ ] **资源管理**
* [ ] **性能优化**
